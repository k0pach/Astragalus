# Astragalus
Пошаговый рогалик. Был вдохновлен игрой Balatro.
Игроку предстоит кидать 5 костей и из выпавших костей собирать комбинации и таким образом набирать очки. Цель игрока набрать нужное количество очков для прохождения уровня.

## Что сделано:
- Регистрация и авторизация;
- БД, где хранятся данные пользователей;
- Геймплей;
- Коллекции (окно, где игрок может посмотреть его коллекцию скинов игральных костей);
- Настройки;
- Проведено Unit-тестирование.
